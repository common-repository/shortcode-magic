== Description ==

A nice and very simple plugin full of amazing shortcodes to use. No coding knowledge required.

A full list of shortcodes can be found on our [Github Wiki page](https://github.com/678Studios/shortcode-magic/wiki/Shortcodes), or in the admin panel. I (i678) orignally made this as a hobby, and decided to release it to the public. If there are any issues, you may report it on the issues tab of the Github repo.

Features:

- Nice Shortcodes

- A nicely designed Admin Panel

== Installation ==

1. Upload the file at wp-content/plugins/
1. Activate the plugin in your admin panel
1. Done!
